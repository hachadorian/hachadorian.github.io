Lab 1 :

The first lab I completed for this class is this website. Prior to this class I have had some experience with HTML/CSS as well as JS. I worked through some of the Odin project which is where I first got this experience. Some of the major CSS techniques involved embedding fonts and images. I think the most important part about this is making sure to use relative paths so that these pages can be deployed easily. I also used some animations with certain divs to make the page more interesting. I also used linear-gradient across this site a number of different times as well. The main thing I did not like is just getting used to properly coding HTML and CSS. Prior to this I definately formed some bad habits that were sort of hard to break. I liked being able to create a site from nothing and being able to design the site as well.
